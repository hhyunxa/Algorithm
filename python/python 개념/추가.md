슬라이싱은 가져옴. 복사해서 가져옴. ex) a[1:3]
그냥 일반 a[1]은 바로 가져오는거임.

슬라이싱이 말이 안되면 빈 문자열 출력.

리스트는 내용 수정, 추가, 삭제 가능. 즉 가변자료형.
재할당은 내용은 같아도 다른 메모리 주소를 가짐.
재할당은 메모리주소를 하나 더 만드는 것. 박스를 하나 더 만듦. # 불변자로형은 재할당으로 해야함.
수정은 박스는 새로 안만듦. 기존 박스에서 값만 수정. #가변자료형만 가능.
```python
a = [1,2] # 리스트
b = [1,2]
print (a is b) # false
```

튜플은 한번 생성되면 그 내용을 수정, 추가, 삭제 불가능. 즉 불변자료형.
파이썬이 내부동작 할 떄 주로 사용. 우리가 직접 쓸일은 거의 없음.
```python
a = (1,2) # 튜플
b = (1,2)
print (a is b) # true
```

range() : 숫자 시퀀스 생성. 주로 for문과 함께 사용. 
range(start, stop, step)
start : 시퀀스 시작 숫자. 기본값 0.
stop : 시퀀스 끝 숫자(포함 안됨).
step : 시퀀스 증가폭. 기본값 1.
list로 형변환 시 내부 값 확인 가능. 
```python
my_range = range(5) 
print(list(my_range)) # [0, 1, 2, 3, 4] 
print(my_range) # range(0, 5)
```

딕셔너리는 키-값 쌍으로 이루어진 자료형. 중괄호 {}로 표현.
키는 고유해야 하며 변경 불가능한 자료형(문자열, 숫자, 튜플 등)만 사용 가능.
값은 어떤 자료형도 가능.
딕셔너리는 순서가 없으며, 인덱싱이나 슬라이싱 불가능. 키를 통해 값에 접근. 즉 비시퀀스임.
```python
d = {
    'name' : 'kyw',
}
a = 'name2'
print(d.get(a)) # None
print(d(a)) # 프로그램 터짐
```

파이썬 표준 라이브러리 (PSL, Python Standard Library)
파이썬을 설치하면 자동으로 사용할 수 있는 기본 라이브러리
- 다양한 기능이 내장되어 있어 복잡한 작업도 쉽게 처리할 수 있음.
- math, random, sys(모듈) 및 json, email(패키지) 등 다양한 모듈과 패키지가 포함됨.
- 별도 설치 없이 바로 import 해서 사용 가능.

파이썬 외부 패키지(Third-party Packages)
필요한 기능을 사용하기 위해 직접 설치해서 쓰는 패키지
- 전 세계 개발자들이 만든 다양한 패키지들이 존재
 ex) 엑셀 파일 조직(pandas, openpyxl) / 데이터 시각화(matplotlib) / 웹 데이터 수집(requests) 등
- 사용할 패키지를 설치할 때는 pip 명령어를 사용

가변객체 : 크기가 큰 데이터를 효율적으로 수정할 수 있음. 기존 객체를 직접 수정 가능.
불변객체 : 메모리 사용을 최소화. 여러 변수가 동일한 객체를 안전하게 공유 가능.

```python
str1 = 'abc' # 를 'adc'로 바꾸기

# 1. 쪼개서
print(str1[:1] + 'd' + str1[2:])

# 2. 다 분해했다가 join
list_str1 = list(str1)
list_str1[1] = 'd'
print(''.join(list_str1))
```
