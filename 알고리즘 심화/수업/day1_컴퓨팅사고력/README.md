## 0. 서론
- 명제 : 참과 거짓으로 판별할 수 있는 문장
  - 2는 짝수이다. -> 참인 명제
  - 3은 짝수이다. -> 거짓인 면제
  - 서울은 대한민국의 수도이다. -> 참인 명제
  - 강의는 재밌다. -> 명제가 아니다.
  - 강사님은 멋있다. -> 명제가 아니다.

- 명제는 보통 이런식으로 많이 사용된다.
  - p -> q (조건명제)
    - p가 참일 때, q도 참이다.

- 예시
- 4가 짝수일 때, 4^2 도 짝수이다.

### 명제를 증명하는 방법

#### 1. 대우 증명
- 대우명제: 4^2이 홀수라면, 4는 홀수이다.
  - 가정이 거짓 -> 무조건 참이다.
  - 대우명제가 참이므로, 원래 명제도 참이다.

#### 2. 직접 증명
- 4가 짝수다 = 4 = (2k)
 -> 4^2 = (2k)^2 = 4(k^2)
 -> 4(k^2) 은 4로 나누어 떨어진다 => 짝수다.

- 귀납법 (대표적인 직접 증명)
  - 특정 사례로 부터 출발해서, 일반적인 규칙을 찾는 방법
    - 기저 사례
    - 귀납 가정
    - 귀납 단계
    - 하나가 되면 끝까지 된다.
  - 도미노 예시
    - 모든 도미노는 연속으로 쓰러진다.
      - 기초 단계: 첫 번째 도미노를 손으로 민다 -> 쓰러짐
      - 귀납 가정: 임의의 자연수 n에 대해, n번째 도미노가 쓰러졌다고 가정하자. 즉, P(n)이 참이라고 가정한다.
      - 귀납 단계: n번째 도미노가 쓰러졌다고 하자.
        - (즉, 𝑃(𝑛)이 참이라고 가정)
        - 도미노는 충분히 가깝게 배치되어 있어서 하나가 쓰러지면 반드시 다음 것도 쓰러진다.
        - 따라서 n번째가 쓰러지면 -> n+1번째도 쓰러진다.
      - 결론: 1번이 쓰러지고 쓰러진 것은 다음 것을 쓰러뜨리므로 모든 도미노가 쓰러진다.
  - [주의] 경험적 귀납법 예시
    - 명제: n일 연속으로 하루 1번 더 연락하면, 상대의 호감도는 n단계 올라간다.
    - 기초 단계: 첫째 날, 평소보다 한 번 더 연락 -> 호감도 상승 (1단계 상승)
      - 즉, P(1)이 참이라고 관찰했다.
    - 귀납 가정: 임의의 n일 동안 매일 한 번 더 연락하면 호감도가 n단계 올라갔다고 가정하자. 즉, P(n)이 참이라고 가정한다.
    - 귀납 단계
      - 첫째 날, 평소보다 한 번 더 연락 -> 호감도 상승
      - 둘째 날, 평소보다 한 번 더 연락 -> 호감도 상승
      - n일 연속 연락 -> n일 연속 호감도 상승

#### 3. 간접 증명 (모순 증명)
- 반대 명제 : 4^2 이 홀수라고 가정하자.
  -> 4^2 = 4(k^2) -> 이건 짝수인데 ...?
  -> 반대명제은 모순이 발생하구나!
  -> 원래 명제가 참이다.

- 귀류법 (대표적인 간접 증명)
  - 결론이 거짓이라고 가정했을 때 모순이 발생하면, 원래 결론은 참이다.
    - 아니라고 하면 말이 안 된다.
  - 예시
    - ```
        명제 P:
        강사님은 와이프를 사랑한다.

        가정:
        강사님은 와이프를 사랑하지 않는다.

        그런데,

        매일 설거지를 한다.

        결혼기념일을 절대 잊지 않는다.

        퇴근 후 직행 귀가한다.

        이 모든 행동이 "사랑하지 않는다"와 양립 가능한가?
        -> 상식적으로 모순

        따라서 가정이 틀림
        -> 강사님은 와이프를 사랑한다
